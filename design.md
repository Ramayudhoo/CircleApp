# Circle App Design

## Overview

A Threads-like social media application.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Express
- Prisma
- PostgreSQL

## Pages

### Login

- Email or username input
- Password input
- Login button
- Link to Register page

### Register

- Full name
- Username
- Email
- Password
- Register button

### Home Feed

- Sidebar navigation
- Create thread form
- Thread list
- Like button
- Comment button

### Profile

- Profile picture
- Bio
- User threads
- Edit profile button

## Design Style

- Minimalist
- Black and white theme
- Rounded cards
- Clean typography
- Responsive layout
